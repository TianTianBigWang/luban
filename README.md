# Luban
```Luban``` Config Build Center

# Operation

### Start Server
```shell script
luban server [`command -c config/config.yaml`]
```

### Migration
```shell script
luban migrate [`command -c config/config.yaml`]
```

### Config Generate
```shell script
luban config new [`command -c config/config.local.yaml`]
```

### User Operation
```shell script
luban user add [username] [password]
```
